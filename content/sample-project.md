# Introduction

Welcome to my new premium gallery portfolio. This project was built to showcase my work in the best possible light.

## Technical Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS + Aceternity UI
- **Content:** Markdown + gray-matter + react-markdown
- **Deployment:** GitHub Pages (Static Export)

### Key Features

1. **Apple Cards Carousel:** A smooth, responsive carousel for navigating projects.
2. **Markdown Rendering:** Clean, readable project details using `@tailwindcss/typography`.
3. **Static Generation:** Fast, SEO-friendly output via `output: export`.
