# Realtime Chat App

Slack과 Discord에서 영감을 받아 개발한 실시간 메시징 플랫폼입니다.

## 주요 기능

- WebSocket 기반 실시간 메시지 전송
- 채널 생성 및 멤버 초대
- 이미지 및 파일 업로드
- 메시지 읽음 확인 및 반응(이모지)

## 기술 스택

- **Frontend:** React, TypeScript, Zustand
- **Backend:** NestJS, Socket.io
- **Database:** PostgreSQL, AWS S3
