# Dev Tools Dashboard

개발 과정에서 분산되어 있던 도구들을 하나의 화면에서 관리할 수 있는 대시보드입니다.

## 주요 기능

- 실시간 서버 로그 스트리밍
- API 요청/응답 테스트 인터페이스
- 성능 메트릭 시각화
- 환경변수 관리 패널

## 기술 스택

- **Frontend:** Next.js, Tailwind CSS, Recharts
- **Backend:** Node.js, WebSocket
- **Database:** Redis (캐싱)
