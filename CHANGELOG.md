# Changelog

## [v2.0.0] - 2026-02-27
### Added
- **인터랙티브 Orb 배경**: WebGL(OGL) 기반의 동적 글로우 효과 도입 (Home Page).
- **현대적 포트폴리오 구조**: `content/` 폴더 기반의 Markdown 프로젝트 관리 시스템 구축.
- **다크 모드 전용 디자인**: 블랙 배경 위주의 프리미엄 다크 테마 적용.
- **새로운 갤러리 및 프로젝트 상세 페이지**: `GalleryView` 및 슬러그 기반의 동적 프로젝트 상세 화면 구현.
- **IntroOverlay 컴포넌트**: 초기 로딩 및 진입 시 `PrismBackground`를 활용한 몰입형 도입부 추가.

### Changed
- **Tailwind CSS v4 마이그레이션**: 차세대 스타일 엔진으로 전환 및 최적화.
- **Next.js 15 업그레이드**: 최신 App Router 아키텍처 및 성능 최적화 반영.
- **파일 구조 단순화**: 레거시 컴포넌트 및 중복된 유틸리티 코드 제거.

### Fixed
- **배경 레이아웃 이슈**: Orb 컴포넌트의 위치 및 레이어 순서(z-index) 정렬 문제를 Playwright 테스트를 통해 최종 해결.

---
*이전의 블로그 및 포트폴리오 데이터는 `legacy/v1` 브랜치에 보존되어 있습니다.*
